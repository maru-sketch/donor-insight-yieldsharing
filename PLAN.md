## 유형별 설계
- 단일 화면: 소개·개인정보 안내 → 업로드 → 요약 지표 → 두 개의 반응형 차트 → 편집 가능한 전략 인사이트 → 내보내기
- 입력 데이터는 날짜, 납부 항목, 인입 채널, 금액 열만 집계하며 성명·연락처는 읽어 표시하지 않는다.
- 외부 모델 자격 증명이 없는 첫 버전은 브라우저 내 통계 패턴 분석으로 초안을 만들고 이를 UI에서 명시한다.
- 리포트 초안만 해당 기기의 브라우저 저장소에 저장하며 원본 행은 저장하지 않는다.

## 가정과 결정
- 엑셀 직접 파싱은 첫 구현 포함 목록이 CSV로 한정되어 있어 제외하고, 업로드 도움말에서 CSV 추출 방법을 안내한다.
- “AI 기반”은 외부 AI로 데이터를 전송하지 않는 브라우저 내 패턴 분석 프로토타입으로 구현하며 실제 처리 방식을 숨기지 않는다.
- “업무 시간 50% 단축”은 기준선·관찰 기간이 없어 `FACT_UNVERIFIED` 목표로 유지하고 제품 기능 완료의 통과 증거로 사용하지 않는다.
