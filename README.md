# 후원자 데이터 인사이트 대시보드

후원 CSV를 브라우저에서 분석해 납부 흐름, 인입 채널, 모금 전략 초안을 보여 주는 열매나눔재단 모금 담당자용 웹앱입니다.

## 실행

Node.js 22.13 이상과 pnpm이 필요합니다.

```bash
pnpm install
pnpm dev
```

검증은 `pnpm lint`와 `pnpm test`로 실행합니다.

## CSV 형식

필수 열은 `납부일`, `납부항목`, `인입채널`, `납부금액`입니다. 영문 열 이름(`date`, `item`, `channel`, `amount`)도 인식합니다. UTF-8과 한글 CSV(CP949)를 자동 판별하며, 열 순서를 바꾸거나 불필요한 열을 미리 삭제하지 않아도 필수 열만 자동 연결합니다. 최대 파일 크기는 5MB입니다.

인입채널 값이 비어 있는 행은 분석에서 버리지 않고 `채널 미입력`으로 따로 집계합니다.

후원자 유지 현황은 선택 열인 `회원번호`, `시작년월`, `납부종료일`이 있을 때 계산합니다. 파일의 최신 납부일을 기준일로 삼아 종료일이 없거나 기준일과 같거나 이후인 고유 후원자의 비율을 표시하고, 시작년월부터 중단일 또는 기준일까지의 평균 후원기간도 함께 보여 줍니다.

실제 납입 현황은 선택 열인 `회원번호`, `납부여부`가 있을 때 계산합니다. 회원번호로 중복 인입을 묶은 뒤 납부여부가 `Y`인 기록이 한 번이라도 있는 후원자를 실제 납입으로 보며, 1회 이상 납입률과 실제 납입자·미납입 인입 수를 함께 보여 줍니다.

후원자 성명과 연락처 열이 있어도 분석에서 제외합니다. 회원번호는 중복 후원자 구분에만 브라우저 메모리에서 사용하며 화면·저장·리포트에 포함하지 않습니다. 원본 파일과 분석 행은 서버에 올리거나 저장하지 않으며, 편집한 인사이트 문구만 현재 브라우저의 로컬 저장소에 저장합니다.

## 주요 기능

- CSV 업로드와 형식 오류 안내
- 월별 납부금액·인입 채널별 빈도 시각화
- 연월 × 인입채널별 건수와 월 합계 교차표
- 납부일·납부항목·인입채널·납부금액 중 원하는 열만 고르는 데이터표
- 평균 후원 유지율·평균 후원기간·중단 후원자 수
- 인입 후 1회 이상 실제 납입률·납입자·미납입 인입 수
- 데이터 패턴 기반 전략 인사이트 초안
- 인사이트 문구 편집·브라우저 저장
- 텍스트 리포트 내보내기
- 모바일 반응형 화면

요구사항과 TEST 연결, 검증 증거는 `PLAN.md`에서 확인할 수 있습니다.
